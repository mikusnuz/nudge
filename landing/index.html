Features

Built for Speed

Pure Swift, zero dependencies, minimal footprint.

Keyboard Shortcuts

19 customizable shortcuts for every layout. Fully remappable in settings.

Drag to Edge

Drag any window to the screen edge. A preview overlay shows exactly where it will land.

Multi-Monitor

Press the same shortcut twice to move windows across displays with smart mirroring.

Ignore Apps

Exclude specific apps from window management. Right from the menu bar.

Lightweight

Pure Swift and AppKit. No Electron, no frameworks, no background services eating your RAM.

Open Source

MIT licensed. No tracking, no analytics, no data collection. Inspect every line of code.
