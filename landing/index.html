Features

Simple. Powerful. Free.

Everything you need for window management, nothing you don't.

Keyboard Shortcuts

19 customizable shortcuts for every layout. Halves, quarters, thirds, two-thirds, and more.

Drag to Edge

Drag any window to the screen edge and see a preview overlay. Release to snap into place.

Multi-Monitor

Press the same shortcut twice to send windows across displays. Works with any number of monitors.

Customizable

Remap any shortcut to your preference. Ignore specific apps. Enable or disable drag-snap.

Lightweight

Pure Swift. No Electron. No dependencies. Minimal memory and CPU usage.

Open Source

MIT licensed. Inspect the code, contribute, or fork it. No tracking, no analytics, no data collection.
